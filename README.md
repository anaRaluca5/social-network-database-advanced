# Social Network Database - Advanced (MySQL)

## Description

This project extends a relational database for a social network by incorporating advanced MySQL features such as stored procedures, views, functions, indexing, and full-text search.

## Features

* User management system
* Post creation with relational integrity
* Friendship system with request/accept/reject states
* Data integrity using constraints (FOREIGN KEY, UNIQUE, CHECK)

## Advanced Features

* Stored Procedures for CRUD operations
* Function to compute total number of friends
* Views for simplified data access
* Indexing for performance optimization
* FULLTEXT search for efficient text queries

## Technologies

* MySQL

## Project Structure

* `tema2varfinal.sql` → advanced database schema with procedures, views, and functions

## Key Concepts

* Relational database design
* Data integrity constraints
* Query optimization
* Stored procedures and functions
* Full-text search implementation

## Example Functionalities

* Insert, update, and delete users using stored procedures
* Retrieve simplified user information using views
* Compute the number of friends for a user
* Perform text-based search on posts and user data
